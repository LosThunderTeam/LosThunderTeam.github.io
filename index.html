Key Features


Multiplayer

This mode allows you to go head-to-head with other players as you battle it out to be victorious.

Quiz

This educational aspect of the game allows players to build upon their knowledge as well as learn new information. Answering the question is the key to unlocking the doors

Leaderboard

This element adds a competitive nature to the game. I allows members to be able to view high scores and have a goal of being number 1 on the list.

How it works

With the dedicated controls, each player will be able to naviagate through the various obstacles along the way with the aim to reach the door first. Once at the door, the game will establish if the player has collected the set amount of coins before the player is allowed to face the questions. The player will face a series of questions which will determine whether a player comes victorious or not. However, reaching the door first doesn't guarantee victory. This is because the other player will eventually arrive at the door, who also has a set of questions to answer. The level is won by the player who answers all thier questions in the fastest time. This educational aspect of the game is very important because it challenges the players as well as adding the fun element to the game. The game also contians a leaderboard section which displays a ranking of top players.
